Minha experiência

Como Desenvolvedor Web Front-end e UI Designer, meu objetivo é construir experiências visuais que não só encantam, mas também são plenamente acessíveis. Busco harmonizar estética e funcionalidade para criar interfaces que ofereçam uma interação intuitiva e inclusiva. Aqui você pode visualizar minha experiência profissional:

Infobase IInterativa

UX/UI Designer Pleno

Meu dia a dia é dinâmico e colaborativo. Iniciamos os projetos com uma fase intensa de pesquisa, onde mergulhamos nas necessidades dos usuários e nas particularidades do setor de cada cliente. Seguindo os princípios do design thinking, conduzimos workshops de ideação para esboçar soluções inovadoras. A fase de prototipação é especialmente gratificante, pois é quando transformamos ideias abstratas em interfaces concretas, acessíveis e inclusivas.

Cada projeto é uma nova oportunidade para aplicar e expandir meu conhecimento em design acessível, garantindo que as soluções não apenas atendam aos requisitos visuais e funcionais, mas também promovam uma experiência inclusiva para todos os usuários. Este compromisso com a acessibilidade está no cerne de nosso processo de design, refletindo nossa dedicação a criar produtos que não só pareçam bons, mas que também sejam universalmente utilizáveis.
